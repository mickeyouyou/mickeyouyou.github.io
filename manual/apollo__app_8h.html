class  apollo::common::ApolloApp
 The base module class to define the interface of an Apollo app. An Apollo app runs infinitely until being shutdown by SIGINT or ROS. Many essential components in Apollo, such as localization and control are examples of Apollo apps. The APOLLO_MAIN macro helps developer to setup glog, gflag and ROS in one line. More...

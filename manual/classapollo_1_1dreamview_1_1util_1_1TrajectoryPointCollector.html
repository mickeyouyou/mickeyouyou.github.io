Detailed Description

A utility class that helps simplify the process of adding trajectory points to the SimulationWorld object. The logic is that whenever a new trajectory point is collected, it will create a trajectory segment with its predecessor (except for the first-ever collected trajectory point, obviously, who has no predecessor). The geometry properties of this trajectory segment is then recorded in the SimulationWorld object.
